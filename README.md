# 🌍 AI Trip Architect

AI Trip Architect is a multi-agent travel planning system designed for your upcoming travel landscape. Powered by a specialized **Visa Auditor**, it uses real-time data to craft itineraries while ensuring legal compliance for your specific passport and residency.

## 🚀 Features

- **Consultative Planning:** You will be interviewed about your group size, timing, and pace to ensure a perfect fit.
- **Dynamic Visa Auditing:** A dedicated agent that checks 2026 entry requirements based on your **Citizenship AND Residency**.
- **Real-time Search:** Integrated with SerpApi to find actual flight schedules, hotel availability, and local activities.
- **Orchestrated Intelligence:** An underlying router that seamlessly switches between logistics planning and visa auditing.

## 🛠️ Tech Stack

- **Language:** Python 3.12+
- **Package Manager:** [uv](https://github.com/astral-sh/uv) (Fastest Python resolver)
- **Orchestration:** LangChain v1.0 & LangGraph
- **LLM:** OpenAI GPT-4o-mini
- **UI:** Streamlit
- **Tools:** SerpApi (Google Search Engine)
